## Introduction
SNRV is the national company for recovery and valorisation. They needed a website to showcase their services, products and units, with an admin panel to manage content and images displayed

## Overview
The goal in this project to create an admin panel to control any content (Heading, paragraph, description) and images shown on the website without a need for a developer. We used Filament that provide us with an admin panel ready to use, Laravel with Blade was the main stack for the backend and frontend stack

## Challenges
- How to link each section of the webpage with the right block of data that need to be displayes

## Outcomes
- I learned how to create a simple CMS to manage data

## Technologies
Laravel, Filament, CSS, Bootstrap, MySQL

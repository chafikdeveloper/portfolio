## Introduction
A web-based email platform inspired by Gmail, featuring AI-powered summarization and email generation based
on user prompts.

## Overview
In this project we had the frontend ready provided by the client, our job was to develop the backend and link them together and make the necessary changes. We used Laravel with PostgreSQL database. For payment we went with Paddle Payment Gateway.

## Challenges
- Make a scalable code using the **Adapter** pattern design to handle all provider integration
-  Work with Gmail API

## Outcomes
- Code using the Adapter pattern design
- Work with Gmail API
- Integrate Paddle Payment Gateway
- Link frontend with backend using APIs

## Technologies
Laravel, React, TailwindCSS, PostgreSQL, Docker
